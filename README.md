# Session 01 — Revolut Analyzer

Premier script Python de ma série d'apprentissage.

## Ce que ça fait
Lit un export CSV Revolut et calcule nombre, total, moyenne, et plus grosse dépense.

## Concepts appris
- Variables, types (string, int, float)
- Listes et accès par index
- Boucles for et itération sur fichiers CSV
- Conditions if et filtres
- Accumulateurs et tracking de max avec label
- Conversion float, abs, round

## Exécuter
python3 analyseur.py

Note : revolut.csv n'est pas commité (données personnelles).
